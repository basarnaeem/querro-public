# Screenshots

This folder hosts visual documentation of the Querro platform — landing page, dashboard, questionnaire interface, briefing memo output. Images are added as production milestones are reached and are referenced from the documents in [`/docs`](../docs).

The platform is in active development. Screenshots reflect the current production state at [querro.app](https://querro.app); for the most current view of any flow, the live site is the source of truth.

## How screenshots are captured

Each screenshot in this folder follows three rules.

**1. Real data, sanitized.** Screenshots are taken against working test fixtures, not against mocked or invented data. Where a screenshot includes data that could identify a specific firm or transaction, that data is replaced with neutral placeholder values. The platform's interaction with real data is faithful; the values shown are not.

**2. Annotated for clarity.** Where a screenshot shows a non-obvious interaction or a feature whose purpose is not visible at first glance, the image is annotated with brief callout text. Annotations are added in a consistent style and avoid covering the underlying interface.

**3. Versioned.** Each screenshot's filename includes the production date in `YYYY-MM-DD` form so that readers can place it on the development timeline described in [Roadmap](../docs/06-roadmap.md). Replaced screenshots are removed rather than retained, to keep this folder a current view rather than a historical archive.

## What this folder is not

This folder does not host marketing creative, brand assets, logo files, or design system documentation. Those live separately and are not part of the public documentation set. If you need brand assets — for press, partnership, or fellowship coverage — the contact channel is in [CONTRIBUTING.md](../CONTRIBUTING.md).
